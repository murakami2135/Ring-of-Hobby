Ring of Hobby

サイト概要

最近では、趣味を持たない人が多いようなので 少しでも興味を持ったことを検索しでき、話を聞いたり、 情報の交換ができるアプリケーションを製作しました。 

サイトテーマ
同じ趣味の人と交流できるサイト  

テーマを選んだ理由  
経験者からいろいろ聞いたりして、趣味の幅を広げられたら良いなと考えました。  ### ターゲットユーザ  趣味の幅を広げたい人、今趣味にしている物での繋がりを広げたい人  

主な利用シーン  
家でもできることに自分に無い意見や団体で行う物を探したい方に利用していただく
